安琪笔记：Linux配置java环境

Sep 18, 2019点击

监督安琪学习，她写的第二篇笔记

第一步 下载

下载Linux的jdk .tar.gz

第二步 打开终端

解压后打开终端；也可打开终端使用解压口令：
tar -zxvf jdk…….

第三步 获取权限

为防止无法写入环境变量，我们需获得root权限再写入
sudo vim /etc/profile
输入i进行编辑，编辑完点击ESC，然后 :wq 保存
JAVA_HOME=/home/bear/install/jdk-10.0.2（输入解压的jdk路径，查看路径:pwd）
CLASSPATH=.:$JAVA_HOME/bin.tools.jar
PATH=$JAVA_HOME/bin:$PATH
export JAVA_HOME CLASSPATH PATH

第四步 保存

保存之后配置立即生效
source /etc/profile

第五步 检查

检查配置是否成功
java 或者 javac
